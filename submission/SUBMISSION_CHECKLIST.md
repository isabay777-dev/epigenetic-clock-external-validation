# Submission Checklist

- Insert the final submission date in `submission/COVER_LETTER.md`.
- Run iThenticate, Turnitin, or the journal-required similarity check before submission.
- Create the Zenodo deposit and paste the final DOI into `CITATION.cff` and the manuscript data-availability statement.
- Push the repository to GitHub after removing any private or temporary files not intended for release.
- Submit `submission/TITLE_PAGE.md` and `submission/BLINDED_MANUSCRIPT.md` as separate files through the Biogerontology Editorial Manager system.
- Confirm the blinded manuscript contains no author name, affiliation, email address, ORCID, identifying acknowledgements, or author-revealing repository URL.
